Welcome to VirtualBank web api

Try this:

/api/questions - returns the list of questions

/api/bundles - returns the list of bundles

/api/products - returns the list of products

GET /api/bundles/advise?age=val&student=val&income=val - returns advise on bundle by given attributes (val is answer Id, see /api/questions)

POST /api/bundles/advise - validates modified bundle (use data from GET action)
